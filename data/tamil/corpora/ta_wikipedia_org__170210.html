மார்கோவ் அமைப்பு

நிகழ்தகவு கோட்பாட்டில், மார்கோவ் அமைப்பு (Markov model) ஒரு வகை கோல அமைப்பு (graphical model) ஆகும். மாறிகளின் கூடிய நிகழ்தகவு கீழ்க்கண்டவாறு குறிக்கப்படும்.

p ( x 1 , … , x N ) = p ( x 1 ) ∏ n = 2 N p ( x n | x n − 1 ) = 1 Z ∏ n = 2 N ψ ( x n , x n − 1 ) {\displaystyle p(x_{1},\ldots ,x_{N})=p(x_{1})\prod _{n=2}^{N}p(x_{n}\vert x_{n-1})={\frac {1}{Z}}\prod _{n=2}^{N}\psi (x_{n},x_{n-1})}

இங்கு Z {\displaystyle Z} ஒட்டுமொத்த சார்பு (partition function) மற்றும் ψ {\displaystyle \psi } வலிமை சார்பு (potential function) ஆகும். இந்த அமைப்பில் அடுத்து வரும் நிலை தற்போதைய நிலையை மட்டுமே சார்ந்திருக்கும். ஒரு குறிப்பிட்ட மாறியின் நிகழ்தகவைக் கண்டறிய தகவல் பரிமாற்றத்தைப் (belief propagation) பயன்படுத்தலாம்.

p ( x i ) = 1 Z ∑ x ¬ i ∏ n = 2 N ψ ( x n , x n − 1 ) {\displaystyle p(x_{i})={\frac {1}{Z}}\sum _{x_{\neg i}}\prod _{n=2}^{N}\psi (x_{n},x_{n-1})}

இத்தகைய அமைப்பில், சில நிலைகள் காணப்படவில்லை என்றால், மறைந்த மார்கோவ் அமைப்பு (Hidden Markov Model, or HMM) நேரிடும். இதன் நிகழ்தகவு எடை கீழ்க்கண்டவாறு குறிக்கப்படும்.

p ( x 1 , … , x N , z 1 , … , z N ) = p ( z 1 ) ∏ n = 2 N p ( z n | z n − 1 ) ∏ n = 1 N p ( x n | z n ) {\displaystyle p(x_{1},\ldots ,x_{N},z_{1},\ldots ,z_{N})=p(z_{1})\prod _{n=2}^{N}p(z_{n}\vert z_{n-1})\prod _{n=1}^{N}p(x_{n}\vert z_{n})}

இங்கு x 1 , … , x N {\displaystyle x_{1},\ldots ,x_{N}} நோக்கிய அளவீடுகள் மற்றும் z 1 , … , z N {\displaystyle z_{1},\ldots ,z_{N}} மறைந்த மாறிகள் ஆகும்.

இயல்நிலைப் பரவலைக் கொண்டு கால்மன் வடிப்பானை அமைப்பைப் பெறலாம்.

பின்வருவனவற்றையும் பார்க்கவும் ஆழ்ந்த வலையமைப்பு (deep belief network) நேரியல் இயக்கவியல் அமைப்பு (linear dynamical system)
