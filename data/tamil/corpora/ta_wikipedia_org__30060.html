களம் (கணிதம்)

களம் (Field) என்பது நுண்புல இயற்கணிதத்தில் ஒரு கணித அமைப்பு. அதனில் கூட்டல், பெருக்கல் என இரண்டு செயல்முறைகளும் அவைகளுக்கு நேர்மாறான கழித்தல், வகுத்தல் என்ற இரண்டு செயல்முறைகளும் இருக்கும். சாதாரண அடிப்படை எண் கணிதத்தில் இருப்பது போன்று அவை ஒன்றுக்கொன்று ஒத்துப்போவதாகவும் (compatible) இருக்கும்.

வரையறைகள்

ஒரு களத்தின் இலக்கணத்தை மூன்றுவிதமாக அறிமுகப் படுத்தலாம்.

1. முற்றொருமை யைக்கொண்ட ஒரு பரிமாற்று வளையம் F இல் தொடங்குவோம். அதனாலேயே அதனில் ‘+’ என்ற ஒரு கூட்டல் செயல்முறையும், ‘*’ என்ற ஒரு பெருக்கல் செயல் முறையும் உள்ளன. மற்றும் கூட்டலுக்கு அது ஒரு பரிமாற்றுக் குலமாகவும் பெருக்கல் ஒரு பரிமாற்றுச் செயல் முறையாகவும் உள்ளன. இதைத் தவிர கூட்டலும் பெருக்கலும் ஒழுங்காகப் பகிர்ந்து கொள்கின்றன. இவ்வளவுக்கும் மேல் F இனுள் சூனியமல்லாத ஒவ்வொரு உறுப்புக்கும் ஒரு பெருக்கல் நேர்மாறு இருக்குமானால் , F ஒரு களம் எனப்படும்.

குறிப்பு: a {\displaystyle a} என்ற உறுப்புக்குப் பெருக்கல் நேர்மாறு என்பது கீழ்க்காணும் பண்புடைய x {\displaystyle x} என்ற உறுப்பு:

a x = x a = 1 {\displaystyle ax=xa=1} .

2. பெருக்கல் பரிமாற்றுச் செயலாக இல்லாமல் இருந்தால், மேற்சொன்ன வரையறையில் தொடங்கப்படும் வளையம் F பரிமாறா வளையமாக இருக்கும். இப்படிப்பட்ட F க்குள் சூனியமல்லாத ஒவ்வொரு உறுப்புக்கும் ஒரு பெருக்கல் நேர்மாறு இருக்குமானால் , F ஒரு பரிமாறாக்களம் (Division ring; Skew Field) எனப்படும். இதற்கும் களத்திற்கும் உள்ள ஒரே வேறுபாடு, பெருக்கலின் பரிமாறல் பண்புதான்.

3. ஒரு களம் (F, +, *) இன் கொற்கோள்கள் எல்லாவற்றையும் அடிமட்டத்திலிருந்து கீழே உள்ளபடி கொடுக்கலாம்( '+': கூட்டல்; '*': பெருக்கல்):

(F1): '+' ஓர் ஈருறுப்புச்செயல்; அ-து, F இல் உள்ள எந்த x , y , {\displaystyle x,y,} க்கும், x + y ∈ F {\displaystyle x+y\in F}

(F2): '+' ஒரு சேர்ப்பு விதி; அ-து, F இல் உள்ள எந்த x , y , z {\displaystyle x,y,z} க்கும் ( x + y ) + z = x + ( y + z ) {\displaystyle (x+y)+z=x+(y+z)}

(F3): '+' ஒரு பரிமாற்று விதி; அ-து, F இல் உள்ள எந்த x , y {\displaystyle x,y} க்கும், x + y = y + x {\displaystyle x+y=y+x}

(F4): F இல் கூட்டலுக்கு ஒரு முற்றொருமை உள்ளது; அ-து, F இல் '0' என்ற ஒரு உறுப்பு கீழேயுள்ள பண்புடன் உள்ளது:

F இல் உள்ள எந்த x {\displaystyle x} க்கும், x + 0 = x = 0 + x {\displaystyle x+0=x=0+x}

(F5): F இல் உள்ள ஒவ்வொரு x {\displaystyle x} க்கும் ஒரு (கூட்டல்) நேர்மாறு உளது; அ-து, ஒவ்வொரு x {\displaystyle x} க்கும் ஒரு ( − x ) {\displaystyle (-x)} கீழேயுள்ள பண்புடன் உள்ளது:

x + ( − x ) = 0 = ( − x ) + x {\displaystyle x+(-x)=0=(-x)+x}

(F6): '*' ஓர் ஈருறுப்புச்செயல்; அ-து, F இல் உள்ள எந்த x , y , {\displaystyle x,y,} க்கும், x ∗ y ∈ F {\displaystyle x*y\in F}

(F7): '*' ஒரு சேர்ப்பு விதி: அ-து, F இல் உள்ள எந்த x , y , z {\displaystyle x,y,z} க்கும் ( x ∗ y ) ∗ z = x ∗ ( y ∗ z ) {\displaystyle (x*y)*z=x*(y*z)}

(F8): '*' ஒரு பரிமாற்று விதி: அ-து, F இல் உள்ள எந்த x , y {\displaystyle x,y} க்கும், x ∗ y = y ∗ x {\displaystyle x*y=y*x}

(F9): F இல் பெருக்கலுக்கு ஒரு முற்றொருமை கூட்டல் முற்றொருமையைவிட வேறானதாக உள்ளது; அ-து, F இல் '1' என்ற ஒரு உறுப்பு ( ≠ 0 {\displaystyle \neq 0} ) கீழேயுள்ள பண்புடன் உள்ளது:

F இல் உள்ள எந்த x {\displaystyle x} க்கும், x ∗ 1 = x = 1 ∗ x {\displaystyle x*1=x=1*x}

(F10): F இல் உள்ள ஒவ்வொரு x {\displaystyle x} க்கும் ஒரு (பெருக்கல்) நேர்மாறு உளது; அ-து, ஒவ்வொரு x {\displaystyle x} க்கும் ஒரு x − 1 {\displaystyle x^{-1}} கீழேயுள்ள பண்புடன் உள்ளது:

x ∗ x − 1 = 1 = x − 1 ∗ x {\displaystyle x*x^{-1}=1=x^{-1}*x}

(F11): கூட்டலும் பெருக்கலும் ஒன்றுக்கொன்று பகிர்ந்துகொள்ளக்கூடியவை; அ-து,F இல் உள்ள எந்த x , y , z {\displaystyle x,y,z} க்கும்

x ∗ ( y + z ) = ( x ∗ y ) + ( x ∗ z ) {\displaystyle x*(y+z)=(x*y)+(x*z)}

( x + y ) ∗ z = ( x ∗ z ) + ( y ∗ z ) {\displaystyle (x+y)*z=(x*z)+(y*z)}

முடிவுறாக்களங்கள்

Q {\displaystyle \mathbf {Q} } என்ற விகிதமுறு எண்களின் கணம்.

R {\displaystyle \mathbf {R} } என்ற மெய்யெண்களின் கணம்.

C {\displaystyle \mathbf {C} } என்ற சிக்கலெண்களின் கணம்.

இவை மூன்றிலும் கூட்டலும் பெருக்கலும் அவைகளில் இயற்கையாகவே உள்ள கூட்டலும் பெருக்கலும் தான்.

இவை மூன்றும் முடிவுறாக்களங்கள்.

முடிவுறு களங்கள்

முடிவுறு களங்களும் உள்ளன. இவைகளைப்பற்றிய கோட்பாடு ஒரு தனிப்பிரிவாகவே விரியும். ஒரே ஒரு எளிதான எடுத்துக்காட்டு கீழே ஒரு அட்டவணையாகக் கொடுக்கப்பட்டுள்ளது. இதனில் ஏழு உறுப்புகளே: 0, 1, 2, 3, 4, 5, 6. கூட்டலும் பெருக்கலும் modulo 7 முறைப்படி செய்யப்படுகின்றன. அதாவது, 5 + 3 = 8 = 1(mod 7). 5 × {\displaystyle \times } 3 = 15 = 1(mod 7).

அட்டவணையைப்பார்த்து நாம் சொல்லி விடலாம். சூனியமல்லாத ஒவ்வொரு உறுப்புக்கும் ஒரு பெருக்கல் நேர்மாறு உள்ளது. எடுத்துக்காட்டாக, 5 − 1 = 3 ; 6 − 1 = 6 {\displaystyle 5^{-1}=3;6^{-1}=6} .

சுருங்கச்சொன்னால் ஒவ்வொரு பகா எண் p க்கும். p உறுப்புகளுள்ள ஒரு தனிப்பட்ட முடிவுறுகளம் உள்ளது. p r {\displaystyle p^{r}} போன்ற பகா எண்களின் மடக்குகள் ஒவ்வொன்றுக்கும் ஒரு களம் உள்ளது. ஆனால் அது அவ்வளவு எளிதானதல்ல. அதற்கு G F ( p r ) {\displaystyle GF(p^{r})} என்ற பெயர். இங்கு GF என்றால் Galois Field. கால்வா (Galois) என்ற இளம் கணித இயலர் 20 வயதுக்குள் பலஆய்வுகள் செய்து தான் துரதிருஷ்டவசமாக ஒரு துப்பாக்கிச் சண்டையில் இறக்கப் போவதை எதிர்பார்த்து இறப்பதற்குமுன் அவர் ஆய்வுகளை எழுதிவைத்துவிட்டுப்போனார்.
