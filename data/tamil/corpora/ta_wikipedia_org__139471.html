-1 (எண்)

மறை ஒன்று அல்லது சய ஒன்று (ஆங்கிலம்: Minus One) என்பது 1 இன் கூட்டல் நேர்மாறு ஆகும். மறை ஒன்று என்பது மறை இரண்டுக்கும் பூச்சியத்துக்கும் இடைப்பட்ட மறை நிறையெண் ஆகும்.

இயற்கணிதப் பண்புகள்

ஓர் எண்ணை, − 1 {\displaystyle -1} ஆல் பெருக்குவது என்பது, அவ்வெண்ணின் குறியை மாற்றுவதற்குச் சமனாகும். இதனைப் பரம்பல் விதி மூலம் நிறுவலாம்.

x {\displaystyle x} ஒரு மெய்யெண் எனில்:-

x + ( − 1 ) ⋅ x {\displaystyle x+(-1)\cdot x}

= 1 ⋅ x + ( − 1 ) ⋅ x {\displaystyle =1\cdot x+(-1)\cdot x} = ( 1 + ( − 1 ) ) ⋅ x {\displaystyle =(1+(-1))\cdot x} = 0 ⋅ x {\displaystyle =0\cdot x} = 0 {\displaystyle =0} ஃ x + ( − 1 ) ⋅ x = 0 {\displaystyle x+(-1)\cdot x=0\,}

x {\displaystyle x} உடன் ( − 1 ) ⋅ x {\displaystyle (-1)\cdot x} ஐக் கூட்டும் போது = 0 {\displaystyle =0} கிடைப்பதால், ( − 1 ) ⋅ x {\displaystyle (-1)\cdot x} என்பது x {\displaystyle x} இன் கூட்டல் நேர்மாறாகும்.

அதாவது: ( − 1 ) ⋅ x = − x {\displaystyle (-1)\cdot x=-x}

மறை ஒன்றின் வர்க்கம்

-1இன் வர்க்கத்தின் பெறுமானம் 1 ஆகும். இதனால் இரு மறை எண்களின் பெருக்கத்தின் பெறுமானம் ஒரு நேர் எண்ணாக அமையும்.[1]

-1 வர்க்கத்தின் பெறுமானம் 1 ஆகும் என்பதை அட்சர கணித முறைப்படி நிறுவலாம்:

0 = − 1 ⋅ 0 {\displaystyle 0=-1\cdot 0}

= − 1 ⋅ [ 1 + ( − 1 ) ] {\displaystyle =-1\cdot [1+(-1)]} − 1 ⋅ 1 + ( − 1 ) ⋅ ( − 1 ) {\displaystyle -1\cdot 1+(-1)\cdot (-1)} = − 1 + ( − 1 ) ⋅ ( − 1 ) {\displaystyle =-1+(-1)\cdot (-1)}

சமன்பாட்டின் இரண்டு பக்கங்களிலும் 1ஐக் கூட்டுவதன் மூலம்,
