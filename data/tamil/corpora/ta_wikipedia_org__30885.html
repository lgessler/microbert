இணையாட்களம் (கணிதம்)

கணிதத்தில், சார்பு f {\displaystyle f} : X {\displaystyle X} → Y {\displaystyle Y} இன் இணையாட்களம் (Codomain) என்பது Y {\displaystyle Y} என்ற கணம்.

f {\displaystyle f} இன் ஆட்களம் என்பது X {\displaystyle X} .

{ f ( x ) {\displaystyle f(x)} : x {\displaystyle x} ∈ X {\displaystyle X} } என்ற கணம் f {\displaystyle f} இன் வீச்சு எனப்படும். அதை f ( X ) {\displaystyle f(X)} என்றும் எழுதலாம்.

இதனிலிருந்து f {\displaystyle f} இன் வீச்சு எப்பொழுதும் அதன் இணையாட்களத்தின் உட்கணமாகத்தான் இருக்கவேண்டும் என்று ஊகிக்கலாம்.

எடுத்துக்காட்டு

இணையாட்களத்திற்கும் வீச்சுக்கும் உள்ள வேறுபாட்டைப் புரிந்துகொள்ள ஒரு நேரியல் உருமாற்றத்தின் அணியைக் கருத்தில் கொள்வது உதவும். வழக்கப்படி, ஒரு m × n {\displaystyle m\times n} அணியுடன் உறவுப்படுத்தப்பட்ட ஒரு நேரியல் கோப்பின் அரசு Rn, மற்றும் இணையாட்களம் Rm. ஆனால் அவ்வணியை எல்லா n-பரிமாண நிரல்திசையன்களால் வலது பக்கம் பெருக்கினால் கிடைக்கும் கணம் மிகச்சிறியதாகவும் இருக்கலாம். உதாரணத்திற்கு, அணியின் எல்லா உறுப்புக்களும் 0 வாக இருந்தால், வீச்சு, அதன் பரிமாணம் m {\displaystyle m} ஆக இருந்தாலும் ஒரு சூனிய அணியே. அணியின் ஏதாவதொரு உறுப்பை மாற்றினால் கூட வீச்சு பெரிதாகிவிடும்.

இன்னொரு எடுத்துக்காட்டு f {\displaystyle f} : R → {\displaystyle \rightarrow } R வரையறை: f : x ↦ x 2 . {\displaystyle f\colon \,x\mapsto x^{2}.}

f இன் இணையாட்களம் R. ஆனால் f(x) எதிர்ம மதிப்புகளை வெளியீடு செய்வதில்லை.அதனால் வீச்சு R 0 + {\displaystyle _{0}^{+}} : , அதாவது எதிர்மமில்லாத மெய்யெண்கள், அதாவது [0, ∞ {\displaystyle \infty } ):

0 ≤ f ( x ) < ∞ . {\displaystyle 0\leq f(x)<\infty .}

இப்பொழுது g {\displaystyle g} என்ற இன்னொரு சார்பை கவனிப்போம்:

g : {\displaystyle g\colon } R → {\displaystyle \rightarrow } R 0 + {\displaystyle _{0}^{+}} : g : x ↦ x 2 . {\displaystyle g\colon \,x\mapsto x^{2}.}

f {\displaystyle f} ம் g {\displaystyle g} ம் ஒரு குறிப்பிட்ட எண்ணை ஒரே மாதிரியாக செயலாக்கினாலும் , தற்காலக்கணிதத்தில் அவையிரண்டும் ஒரே சார்பல்ல. ஏனென்றால் அவைகளினுடைய இணையாட்களங்கள் வெவ்வேறானவை. இதை நன்கறிந்துகொள்ள இன்னொரு சார்பைப்பார்ப்போம்:

h : x ↦ x . {\displaystyle h\colon \,x\mapsto {\sqrt {x}}.}

இங்கு R 0 + {\displaystyle _{0}^{+}} தான் அரசாக வரையறுக்கப்படமுடியும். இப்பொழுது இவ்விரண்டு சேர்வைகளை நோக்குக:

h ∘ f {\displaystyle h\circ f} , h ∘ g {\displaystyle h\circ g} .

இவையிரண்டில் எது சரியான பொருளுள்ளது? முதலிலுள்ளது பிரச்சினையை எழுப்புகிறது. ஏன்? வர்க்கமூலம் எதிர்ம எண்களுக்கு வரையறுக்கப்படவில்லை!

ஆக, சார்புகளின் சேர்வை f 1 ∘ f 2 {\displaystyle f_{1}\circ f_{2}} பேசப்படும்பொழுது, வலது பக்கச்சார்பின் இணையாட்களமும் இடது பக்கத்து சார்பின் ஆட்களமும் ஒரே கணமாக இருக்கவேண்டும்.

இணையாட்களத்தைப்பொருத்துதான் சார்பு ஒரு முழுக்கோப்பா அல்லவா என்பது தீர்மானிக்கப்படும். சார்பு ஒரு உள்ளிடுகோப்பா என்பது இணையாட்களத்தைப்பொருத்ததில்லை.

இவற்றையும் பார்க்கவும் ஆட்களம் (கணிதம்) வீச்சு
