VBScript

Microsoft Visual Basic Scripting Edition (قىسقارتىلمىسى VBS) بولسا Visual Basic ئائىلىسىدىكى يېڭى ئەزا.

ئۇ Microsoft Internet Explorer نىڭ تور ئابۇنتلار بۆلىكىدىكى تايانچ تىل (WEB客户机脚本语言) ۋە Microsoft Internet Information Service (قىسقارتىلمىسى IIS) تور مۇلازىمىتىرى تايانچ تىلى (WEB服务器脚本语言) نى ئۆز ئىچىگە ئالىدۇ.

ئەگەر سىز ئاللىقاچان Visal Basic نى ئۆگىنىپ بولغان بولسىڭىز، ئۇ ھالدا سىزگە VBS نى ئۆگىنىش قىيىنغا توختىمايدۇ، كۆزىڭىزگە نۇرغۇن كودلار تونۇش بىلىنىدۇ، ئەگەر سىز Visual Basic نى ئۆگەنمىگەن بولسىڭىزمۇ چوڭ مەسىلە يوق، چۈنكى VBS ناھايىتى ئاسان ۋە چۈشىنىشلىك تىل.

VBS دا تۆۋەندىكىدەك ئىشلارنى قىلغىلى بولىدۇ.

1. ھېسابلاش

VBS بىزگە يېتەرلىك ماتېماتىكىلىق فۇنكسىيەنى تەمىنلەپ بەرگەن.

2. ھۆججەت ۋە ھۆججەت قىسقۇچ بىر تەرەپ قىلالايدۇ.

3.Windows نى باشقۇرالايدۇ.

4. Word، Excel، PowerPoint ۋە Office ھۆججەتلىرىنى بىر تەرەپ قىلالايدۇ.

مەسىلەن، Word، Excel ھۆججىتى قۇرۇش كودى:

SERVER.CREATEOBJECT("WORD.APPLICATION") WORD SERVER.CREATEOBJECT ("EXCEL.APPLICATION") EXCEL

5. سانلىق مەلۇمات ئامبىرىنى بىر تەرەپ قىلالايدۇ.

6. Windows Media Player نى تەڭشەپ ئىشلەتكىلى ۋە خالىغانچە باشقۇرغىلى بولىدۇ.

7. رەسىم بىر تەرەپ قىلالايدۇ.

8. مۇلازىمىتىر بۆلىكى تېخنىكىسى: Active Server Page (قىسقارتىلمىسى ASP)

9. سىستېما ئىقتىدارلىرىنى تەڭشەپ ئىشلەتكىلى بولىدۇ. ( مەسىلەن ئاۋازلىق پاراڭلىشىش، چايخانا)
