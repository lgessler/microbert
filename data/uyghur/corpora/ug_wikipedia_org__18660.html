پائۇل ھېيزى

گېرمانىيە لىك داﯕﻠﯩﻖ ﻳﺎزﻏﯘﭼﻰ، ﺷﺎﺋﯩﺮ، دراﻣﺎﺗﺮوگ ﭘﺎﺋﯘل ﮬﯧﻴﺰى 1830-ﻳﯩﻞ 15-ﻣﺎرت ﺑﯧﺮﻟﯩﻨﺪا ﺗﯘﻏﯘﻟﻐﺎن. ﺋﻮﺗﺘﯘرا ﻣﻪﻛﺘﻪﭘﺘﻪ ﺋﻮﻗﯘۋاﺗﻘﺎن ﭼﺎﻏﻠﯩﺮﯨﺪﯨﻼ ﺋﻪدەﺑﯩﻴﺎﺗﻘﺎ ﻛﯜﭼﻠﯜك ﺋﯩﺸﺘﯩﻴﺎق ﺑﺎﻏﻼپ، ﺋﯜچ ﺳﺎۋاﻗﺪﯨﺸﻰ ﺑﯩﻠﻪن ﺑﯩﺮﮔﻪ ﺋﻪدەﺑﯩﻴﺎت ﻳﯩﻐﯩﻠﯩﺸﻠﯩﺮﯨﻨﻰ ﺋﻮﻳﯘﺷﺘﯘرۇپ، ﺷﯧﺌﯩﺮ دﯦﻜﻠﻮﻣﺎﺗﺴﯩﻴﻪ ﻗﯩﻠﯩﺶ ﻗﺎﺗﺎرﻟﯩﻖ ﭘﺎﺋﺎﻟﯩﻴﻪﺗﻠﻪر ﺑﯩﻠﻪن ﺷﯘﻏﯘﻟﻼﻧغان

شەخسلەر ئانكىتى

گېرمانىيە لىك داﯕﻠﯩﻖ ﻳﺎزﻏﯘﭼﻰ، ﺷﺎﺋﯩﺮ، دراﻣﺎﺗﺮوگ ﭘﺎﺋﯘل ﮬﯧﻴﺰى 1830-ﻳﯩﻞ 15-ﻣﺎرت ﺑﯧﺮﻟﯩﻨﺪا ﺗﯘﻏﯘﻟﻐﺎن. ﺋﯘﻧﯩﯔ دادﯨﺴﻰ زور ﺷﯚﮬﺮەت ﻗﺎزاﻧﻐﺎن ﺗﯩﻠﺸﯘﻧﺎس، ﺋﺎﻧﯩﺴﻰ داﯕﻠﯩﻖ ﻳﻪﮬﯘدى ﺑﺎﻧﻜﯩﺮﯨﻨﯩﯔ ﻗﯩﺰى بۇلۇپ، ﺋﺎﻧﯩﺴﻰ ﺋﻪدەﺑﯩﻴﺎﺗﻨﻰ ﺳﯚﻳﯩﺪﯨﻐﺎن ﺋﺎﻳﺎل، ﺋﯧﻨﮕﻠﯩﺰ، ﻓﺮاﻧﺴﯘس ﺗﯩﻠﻠﯩﺮﯨﻐﺎ ﭘﯩﺸﺸﯩﻖ بىلەتتى، داۋاﻣﻠﯩﻖ ﺋﻪدەﺑﯩﻲ ﺗﻪرﺟﯩﻤﻪ ﺑﯩﻠﻪن ﺷﯘﻏﯘﻟﻠانغان، ﺑﯘ ﺋﺎﻳﺎﻟﻨﯩﯔ ﺋﺎﭼﺎ- ﺳﯩﯖﯩﻠﻠﯩﺮىدﯨﻦ ﺋﯩﻜﻜﯩﺴﻰ ﺧﯧﻠﻰ ﻧﺎﻣﻰ ﺑﺎر ﻳﺎزﻏﯘﭼﻰ. ﺑﯘ ﺋﻪۋزەﻟﻠﯩﻚ ﭘﺎﺋﯘل ﮬﯧﻴﺰﯨﻨﯩﯔ ﺑﺎﻟﯩﻠﯩﻖ ﮬﺎﻳﺎﺗﯩﻐﺎ زور ﺗﻪﺳﯩﺮ ﻛﯚرﺳﻪﺗﻜﻪن. ﭘﺎﺋﯘل ﮬﯧﻴﺰى 1914-ﻳﯩﻞ 2-ﺋﺎﭘﺮﯦﻞ ﻣﯩﻴﯘﻧﺨﯩﻨﺪا ۋاﭘﺎت ﺑﻮﻟﻐﺎن.

مۇندەرىجە قىسقىچە چۈشەندۈرۈش ئاساسىي ئۇچۇرلىرى شەخسلەر ئانكىتى ئىجتىمائىي پائاليەتلىرى ئەسەرلىرى شان-شەرەپلىرى سۈرەتلەر قامۇس باشبېتى ئىجتىمائىي پائاليەتلىرى

ﭘﺎﺋﯘل ﮬﯧﻴﺰى ﺋﻮﺗﺘﯘرا ﻣﻪﻛﺘﻪﭘﺘﻪ ﺋﻮﻗﯘۋاﺗﻘﺎن ﭼﺎﻏﻠﯩﺮﯨﺪﯨﻼ ﺋﻪدەﺑﯩﻴﺎﺗﻘﺎ ﻛﯜﭼﻠﯜك ﺋﯩﺸﺘﯩﻴﺎق ﺑﺎﻏﻼپ، ﺋﯜچ ﺳﺎۋاﻗﺪﯨﺸﻰ ﺑﯩﻠﻪن ﺑﯩﺮﮔﻪ ﺋﻪدەﺑﯩﻴﺎت ﻳﯩﻐﯩﻠﯩﺸﻠﯩﺮﯨﻨﻰ ﺋﻮﻳﯘﺷﺘﯘرۇپ، ﺷﯧﺌﯩﺮ دﯦﻜﻠﻮﻣﺎﺗﺴﯩﻴﻪ ﻗﯩﻠﯩﺶ ﻗﺎﺗﺎرﻟﯩﻖ ﭘﺎﺋﺎﻟﯩﻴﻪﺗﻠﻪر ﺑﯩﻠﻪن ﺷﯘﻏﯘﻟﻼﻧﻐﺎن.

ﭘﺎﺋﯘل ﮬﯧﻴﺰى 1847- ﻳﯩﻠﻰ ﺋﻮﺗﺘﯘرا ﻣﻪﻛﺘﻪﭘﻨﻰ ﭘﯜﺗﺘﯜرۈپ، ﺑﯧﺮﻟﯩﻦ ﺋﯘﻧﯟﯦﺮﺳﯩﺘﯧﺘﯩﻐﺎ ﻛﯩﺮﯨﭗ ﻛﻼﺳﺴﻚ ﺗﯩﻠﺸﯘﻧﺎﺳﻠﯩﻖ ﻛﻪﺳﭙﯩﺪە ﺋﻮﻗﯘﻏﺎن، ﺑﯩﺮاق ﭘﯜﺗﯜن ۋۇﺟﯘدى ﺑﯩﻠﻪن ﺋﻪدەﺑﯩﻴﺎت ﺋﯚﮔﯩﻨﯩﺸﻜﻪ ﺗﯘﺗﯘش ﻗﯩﻠﻐﺎن. 1848- ﻳﯩﻞ ﻣﺎرﺗﺘﺎ ﻳﺎۋروﭘﺎدﯨﻜﻰ ﮬﻪرﻗﺎﻳﺴﻰ دۆﻟﻪﺗﻠﻪردە ﺧﻪﻟﻖ ﺋﯩﻨﻘﯩﻼﺑﻰ ﻛﯚﺗﯜرۈﻟﮕﻪﻧﺪە، ﮔﯧﺮﻣﺎﻧﯩﻴﯩﺪﯨﻤﯘ ﺋﯩﻨﻘﯩﻼپ ﭘﺎرﺗﻠﯩﻐﺎن، ﺋﯘ ﺗﻪﺷﻪﺑﺒﯘﺳﻜﺎرﻟﯩﻖ ﺑﯩﻠﻪن ﺋﯩﻨﻘﯩﻼﭘﻘﺎ ﻗﺎﺗﻨﺎﺷﻘﺎن ﮬﻪﻣﺪە ﺋﯩﻨﻘﯩﻼﭘﻨﻰ ﻣﻪدﮬﯩﻴﯩﻠﻪﻳﺪﯨﻐﺎن ﺷﯧﺌﯩﺮﻻرﻧﻰ ﻳﺎزﻏﺎن.

ﺋﯩﻨﻘﯩﻼپ ﻣﻪﻏﻠﯘپ ﺑﻮﻟﻐﺎﻧﺪﯨﻦ ﻛﯧﻴﯩﻦ، ﭘﺎﺋﯘل ﮬﯧﻴﺰى ﺑﻮﻧﯩﻦ ﺋﯘﻧﯟﯦﺮﺳﯩﺘﯧﺘﯩﻐﺎ ﻛﯩﺮﯨﭗ ﺋﻮﻗﯘﻏﺎن، 1850- ﻳﯩﻠﻰ ﻳﺎۋروﭘﺎ ﺳﻪﭘﯩﺮﯨﻨﻰ ﺑﺎﺷﻼپ، ﻛﻼﺳﺴﯩﻚ ﺗﯩﻠﺸﯘﻧﺎﺳﻠﯩﻖ (ﻻﺗﯩﻦ ﺗﯩﻠﻰ، ﮔﯧﺮﯨﻚ ﺗﯩﻠﻰ) ﻛﻪﺳﭙﯩﺪﯨﻦ ۋاز ﻛﯧﭽﯩﭗ، روﻣﺎن ﺗﯩﻠﻰ (ﺋﯩﺘﺎﻟﯩﻴﺎن، ﻓﺮاﻧﺴﯘس، ﺋﯩﺴﭙﺎن ﺗﯩﻠﻠﯩﺮى) ﺋﯚﮔﯩﻨﯩﺸﻜﻪ ﻛﯩﺮﯨﺸﻜﻪن. 1852- ﻳﯩﻠﻰ ﭘﺮۇﺳﺴﯩﻴﻪ ﮬﯚﻛﯜﻣﯩﺘﯩﻨﯩﯔ ﻣﻪﺑﻠﻪغ ﭼﯩﻘﯩﺮﯨﺸﻰ ﺑﯩﻠﻪن ﺋﯩﺘﺎﻟﯩﻴﯩﺪە ﺑﯩﺮ ﻳﯩﻞ ﺋﻮﻗﯘﻏﺎن، 1853- ﻳﯩﻠﻰ ﺑﯧﺮﻟﯩﻨﻐﺎ ﻗﺎﻳﺘﯩﭗ ﻛﻪﻟﮕﻪن.

ﺋﯩﺘﺎﻟﯩﻴﯩﺪە ﺋﻮﻗﯘش ﺟﻪرﻳﺎﻧﯩﺪا ﺋﯩﺘﺎﻟﯩﻴﯩﻨﯩﯔ ﻣﻪﻧﺰﯨﺮﯨﺴﻰ، ﺋﯩﺘﺎﻟﯩﻴﯩﻠﯩﻜﻠﻪر ۋە ﺋﯩﺘﺎﻟﯩﻴﯩﻨﯩﯔ ﺗﺎرﯨﺨﯩﻲ ﻣﻪدەﻧﯩﻴﯩﺘﻰ ﺋﯘﻧﯩﯖﻐﺎ ﭼﻮﯕﻘﯘر ﺗﻪﺳﯩﺮ ﻗﯩﻠﻐﺎن ﮬﻪﻣﺪە ﺋﯘﻧﯩﯔ ﺋﯩﺠﺎدﯨﻴﯩﺘﯩﺪە ﺑﯘرۇﻟﯘش ﮬﺎﺳﯩﻞ ﻗﯩﻠﻐﺎن. ﺑﯘ ﻣﻪزﮔﯩﻠﺪە ﺋﯘ ﺗﻪرﺳﺎ «ﻗﯩﺰ» ﻧﻰ ﻳﯧﺰﯨﭗ زور ﻏﯘﻟﻐﯘﻻ ﻗﻮزﻏﯩﻐﺎن.

ﭘﺎﺋﯘل ﮬﯧﻴﺰى داﯕﻠﯩﻖ ﻳﺎزﻏﯘﭼﻰ، ﺷﺎﺋﯩﺮ، دراﻣﺎﺗﺮوگ ﺑﻮﻟﯘپ، 50 ﻳﯩﻠﻠﯩﻖ ﺋﯩﺠﺎدﯨﻴﻪت ﮬﺎﻳﺎﺗﯩﺪا ﺗﻮﻗﻘﯘز روﻣﺎن، 100 دﯨﻦ ﺋﺎرﺗﯘق ﮬﯧﻜﺎﻳﻪ ۋە ﭘﻮۋﺳﺖ، ﻧﻪﭼﭽﻪ ﻳﯜزﻟﯩﮕﻪن ﻟﯧﺮﯨﻚ ۋە ﺳﯩﻴﺎﺳﯩﻲ ﺷﯧﺌﯩﺮﻻرﻧﻰ ﻳﺎزﻏﺎن، 70 ﻛﻪ ﻳﯧﻘﯩﻦ ﺳﻪﮬﻨﻪ ﺋﻪﺳﯩﺮﯨﻨﻰ رۇﻳﺎﭘﻘﺎ ﭼﯩﻘﺎرﻏﺎن. ﺋﯘ ﻳﻪﻧﻪ ﻧﯘرﻏﯘﻧﻠﯩﻐﺎن ﺋﻪﺳﻠﯩﻤﻪ، ﻛﯜﻧﺪﯨﻠﯩﻚ ﺧﺎﺗﯩﺮﯨﻠﻪرﻧﻰ ﻳﺎزﻏﺎن. ﺋﯘﻧﯩﯔ ﺋﻪﺳﻪرﻟﯩﺮﯨﺪە ﺋﯩﺴﺘﯧﺘﯩﺰﯨﻢ ۋە ﻓﻮرﻣﺎﻟﯩﺰﯨﻤﻠﯩﻖ ﺧﺎﮬﯩﺶ ﻛﯜﭼﻠﯜك، ﺋﻪﻣﻤﺎ ﺑﻪدﯨﺌﯩﻲ ﺗﺎﻻﺗﯩﻨﯩﯔ ﻳﯘﻗﯩﺮﯨﻠﯩﻘﻰ ﺑﯩﻠﻪن ﺋﯘ ﺋﻪﺳﻪرﻟﯩﺮﯨﻨﻰ ﻧﺎﮬﺎﻳﯩﺘﻰ ﻣﯘۋەﭘﭙﻪﻗﯩﻴﻪﺗﻠﯩﻚ ﺗﺎﻣﺎﻣﻼپ، زور ﺋﺎﻟﻘﯩﺸﻘﺎ ﺋﯧﺮﯨﺸﻜﻪن.

ئەسەرلىرى

ﭘﺎﺋﯘل ﮬﯧﻴﺰى داﯕﻠﯩﻖ ﻳﺎزﻏﯘﭼﻰ، ﺷﺎﺋﯩﺮ، دراﻣﺎﺗﺮوگ ﺑﻮﻟﯘﭘﻼ ﻗﺎﻟﻤﺎي، ﻣﻪﺷﮭﯘر ﺗﻪرﺟﯩﻤﺎن. ﺋﯘ ﻳﻪﻧﻪ ﺋﯩﺴﭙﺎﻧﯩﻴﻪ، ﻓﺮاﻧﺴﯩﻴﻪ ﺷﺎﺋﯩﺮﻟﯩﺮﯨﻨﯩﯔ ﺷﯧﺌﯩﺮﻟﯩﺮﯨﻨﻰ، ﺋﯩﺘﺎﻟﯩﻴﯩﻨﯩﯔ ﺑﺎﻟﯩﻼر ﭼﯚﭼﻪﻛﻠﯩﺮى، ﺧﻪﻟﻖ ﻧﺎﺧﺸﯩﻠﯩﺮى ۋە ﻛﻮﻣﺪﯨﻴﻪ ۋە ﭘﺮۇزا ﺋﻪﺳﻪرﻟﯩﺮﯨﻨﻰ ﻧﯧﻤﯩﺲ ﺗﯩﻠﯩﻐﺎ ﺗﻪرﺟﯩﻤﻪ ﻗﯩﻠﻐﺎن، ﺋﯘﻧﯩﯖﺪﯨﻦ ﺑﺎﺷﻘﺎ ﺷﯧﻜﯩﺴﭙﯩﺮ ﻗﺎﺗﺎرﻟﯩﻖ داﯕﻠﯩﻖ ﺋﺎدﯨﺒﻠﻪرﻧﯩﯔ ﻣﯘﮬﯩﻢ ﺋﻪﺳﻪرﻟﯩﺮﯨﻨﯩﻤﯘ ﺗﻪرﺟﯩﻤﻪ ﻗﯩﻠﯩﭗ ﺗﻮﻧﯘﺷﺘﯘرۇپ، زور ﺷﯚﮬﺮەت ﻗﺎزاﻧﻐﺎن. ﺋﯘ 1871- ﻳﯩﻠﺪﯨﻦ 1903- ﻳﯩﻠﻐﯩﭽﻪ «ﻧﯧﻤﯩﺲ ﺗﯩﻠﯩﺪﯨﻜﻰ ﭘﻮۋﯦﺴﺖ- ﮬﯧﻜﺎﻳﯩﻠﻪر ﺧﻪزﯨﻨﯩﺴﻰ»، «ﭼﻪت ﺋﻪل ﭘﻮۋﯦﺴﺖ- ﮬﯧﻜﺎﻳﯩﻠﻪر ﺧﻪزﯨﻨﯩﺴﻰ»، «ﻳﯧﯖﯩﺪﯨﻦ ﺗﯜزۈﻟﮕﻪن ﻧﯧﻤﯩﺲ ﺗﯩﻠﯩﺪﯨﻜﻰ ﭘﻮۋﯦﺴﺖ- ﮬﯧﻜﺎﻳﯩﻠﻪر ﺧﻪزﯨﻨﯩﺴﻰ» دﯨﻦ ﺋﯩﺒﺎرەت ﺋﯜچ ﻳﯜرۈش ﻣﯘﮬﯩﻢ ﻛﯩﺘﺎﺑﻨﻰ ﻧﻪﺷﯩﺮ ﻗﯩﻠﺪۇرۇپ، دۇﻧﻴﺎ ﺋﻪدەﺑﯩﻴﺎﺗﯩﻐﺎ زور ﺗﯚﮬﭙﻪ ﻗﻮﺷﻘﺎن. ﺑﯘ ﺋﻪﺳﻪرﻟﻪر 62 ﻛﯩﺘﺎب ﺑﻮﻟﻐﺎن.

ﺋﯘﻧﯩﯔ ۋەﻛﯩﻠﻠﯩﻚ ﺋﻪﺳﻪرﻟﯩﺮﯨﺪﯨﻦ «ﺗﺮاﺑﺒﯩﻠﯩﻚ ﻗﯩﺰ» (1855_ ﻳﯩﻠﻰ)، «ﻏﻪزەپ» (1855- ﻳﯩﻠﻰ)، «ﺋﻪﻧﻨﯩﻨﺎ»(1860 - ﻳﯩﻠﻰ)، «ﺗﺎغ ﭼﻮﻗﯩﻠﯩﺮى ﺋﯜﺳﺘﯩﺪە»، «دۇﻧﻴﺎﻧﯩﯔ ﺑﺎﻟﯩﻠﯩﺮى»، (1873- ﻳﯩﻠﻰ) ﻗﺎﺗﺎرﻟﯩﻘﻼر ﺑﺎر.

شان-شەرەپلىرى

1910-يىلى ﻧﻮﺑﯧﻞ ﺋﻪدەﺑﯩﻴﺎت ﻣﯘﻛﺎﭘﺎﺗﻰ ﺑﺎﮬﺎﻻش ﻛﻮﻣﯩﺘﯧﺘﻰ «ﺋﯘﻧﯩﯔ داﯕﻠﯩﻖ ﻟﯩﺮﯨﻚ ﺷﺎﺋﯩﺮ، دراﻣﺎﺗﺮوگ، روﻣﺎن ﻳﺎزﻏﯘﭼﯩﺴﻰ ۋە دۇﻧﻴﺎدﯨﻜﻰ ﻣﻪﺷﮭﯘر ﭘﻮۋﯦﺴﺖ- ﮬﯧﻜﺎﻳﻪ ﻳﺎزﻏﯘﭼﯩﺴﯩﻐﺎ ﺋﺎﻳﻼﻧﻐﺎﻧﻠﯩﻘﻰ، ﺋﯘزۇن ﻳﯩﻠﻠﯩﻖ ﺋﯩﺠﺎدﯨﻴﻪت ﮬﺎﻳﺎﺗﯩﺪا ﺳﯩﯖﯩﺸﯩﺸﻨﻰ ﻛﯚﯕﯜﻟﺪﯨﻜﯩﺪەك ﺋﯩﭙﺎدﯨﻠﻪپ، ﻳﯜﻛﺴﻪك ﺳﻪﻧﺌﻪت ﺗﺎﻟﻨﺘﯩﻨﻰ ﻧﺎﻣﺎﻳﻪن ﻗﯩﻠﻐﺎﻧﻠﯩﻘﯩﻨﻰ ﺗﻪﻗﺪﯨﺮﻟﻪش ﺋﯜﭼﯜن» ﭘﺎﺋﯘل ﮬﯧﻴﺰﯨﻐﺎ ﻧﻮﺑﯧﻞ ﺋﻪدەﺑﯩﻴﺎت ﻣﯘﻛﺎﭘﺎﺗﯩﻨﻰ ﺑﻪرﮔﻪن.
