گىتخاب

گىتخاب (GitHub) دۇنيادىكى ئەڭ چوڭ يۇمشاق دېتال ئېچىش سۇپىسى بولۇپ، ئۆتكەن يىل مىكروسوفت تەرىپىدىن 7 مىليارد 500 مىليون ئامېرىكا دوللىرىغا سېتىۋېلىنغان. بۇ سۇپىغا دۇنيادىكى داڭلىق پروگراممىلار، نۇرغۇن سىستېما، يۇمشاق دېتال، ئەپلەرنىڭ ئەسلى كودى يىغىلغان. ئەپ ئاچقۇچىلار، پروگراممېرلار بۇ سۇپا ئارقىلىق نۇرغۇن تەكرار ئەمگەكنى ئازايتىپ مەسىللەرنى ھەل قىلالايدۇ؛ ئۆزى يازغان كودنى بۇ سۇپىغا چىقىرىپ كۆپچىلىك بىلەن ھەمبەھرلىنەلەيدۇ. باشقا پروگراممېرلار بىلەن مۇنازىرلىشەلەيدۇ.

GitHub بىر ۋاكالىتەن كود باشقۇرۇش سۇپىسى، خېرىدار دائىرىسى ئىنتايىن كەڭ. ئالما، ئامازون، گوگۇل قاتارلىق خەلقئارالىق پەن-تېخنىكا كارخانىلىرىمۇ GitHub نىڭ خېرىدارلىرى. 2015-يىلدىكى ئىستاستىكىدا GitHub قا تىزىملىتىپ ئىشلەتكەن پروگراممېرلار 9 مىليوندىن ئېشىپ كەتكەن.
