CSS

قىسقىچە چۈشەندۈرۈشى: CSS تېخنىكىسى تور بەت ياساشتىكى مۇھىم بولغان تېخنىكا بولۇپ، CSS تېخنىكىسى ئارقىلىق بېكەتنى بىزەش ئىلىپ بارغىلى، بەتنى كۆركەم، ئېچىلىش سۈرئىتى تېز قىلىپ ياساپ چىققىلى بولىدۇ

CSS دېگەن نېمە؟

CSS قەۋەتلىنىش شەكلى دەپمۇ ئاتىلىدۇ، ھەمدە HTML خاسلىقى ياكى بۇيرۇق قاتارلىقلارنى ئىزاھلاشتا ئىشلىتىدۇ، ئۇ يەنە ئۆزىدە ئەسلىدە بار بولغان بىر قىسىم خۇسۇسىيەتلەرنى (مەسىلەن، خەت شەكلى، چوڭ-كىچىكلىكى، رەڭگى قاتارلىقلارنى) كونترول قىلىپلا قالماستىن، بەلكى HTML نىڭ يەنە بىر قىسىمخۇسۇسىيەتلىرىنىمۇ كونترول قىلىدۇ، مەسىلەن ئوبيېكت ئورنى، رەسىم ئۈنۈمى، مائۇس كۆرسەتكۈچى قاتارلىقلار. قەۋەتلىنىش شەكلى ئارقىلىق بىر قېتىمدا كۆپلىگەن تېكىست ھۆججەتلىرىنى كونترول قىلغىلى بولىدۇ، ھەمدە CSS دىكى مەزمۇنلارنى خالىغانچە ئۆزگەرتىپ، تېكىست ھۆججەتلىرىنىڭ شەكلىنى ئاپتوماتىك يېڭىلىغىلى بولىدۇ. كۆچۈرۈپ يۆتكەش ياخشى ئىش ئەمەس .

CSS تېخنىكىسى تور بەت ياساشتىكى مۇھىم بولغان تېخنىكا بولۇپ، CSS تېخنىكىسى ئارقىلىق بېكەتنى بىزەش ئىلىپ بارغىلى، بەتنى كۆركەم، ئېچىلىش سۈرئىتى تېز قىلىپ ياساپ چىققىلى بولىدۇ.
