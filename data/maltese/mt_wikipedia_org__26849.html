M3P

Il-Malta Media Memory Preservation huwa database multimedjali kollaborattiv għall-preservazzjoni tal-memorja fil-media Maltija, operat mill-Fondazzjoni M3P. Oriġinarjament, bejn l-2010 u l-2015 dan id-database kien imħaddem taħt l-isem Malta Music Memory Project, bl-istess isem imqassar ta' M3P.

L-M3P issawwar minn proposta biex jinħoloq database multimedjali kollaborattiv tal-mużika minn Malta u dak kollu assoċjat magħha.[1] Fost diversi inizjattivi matul iż-żmien kien hemm proġett ta' preservazzjon tas-CDs ta' mużika Maltiija.[2]
