Bing

Bing (qabel Live Search, Windows Live Search u MSN Search) hu s-search engine preżenti ta’ Microsoft. Is-servizz ġie żvelat fit-28 ta’ Mejju, 2009 f’San Diego, l-iStati Uniti, u ħareġ għall-ewwel darba fil-pubbliku fit-3 ta’ Ġunju, 2009, b’servizz bi prova jkun disponnibbli mill-1 ta’ Ġunju, 2009.

Suġġerimenti dwar it-tfittxija, u riżulatati mqassma f’kategoriji huma fost l-akbar tibdiliet li s-servizz ta’ Bing ġab fid-dinja tas-search engines. Sa minn Jannar 2010, Bing hu t-tielet l-akbar search engine fuq l-internet, b’ 3.16%, wara Google b’85.35% u Yahoo b’6.15%. Fid-29 ta’ Lulju, 2009, Microsoft u Yahoo ħabbru ftiehim li bih Bing se tmxxi s-servizz tat-tfittxija ta’ Yahoo.

Storja Msn Search

MSN Search kien l-ewwel search engine ta’ Microsoft, li kien jinkorpora search engine, index u web crawler. Is-servizz ġie minedi fit-tielet kwart tal-1998, u uża r-riżultati ta’ Inktomi. Fil-bidu tal-1999, MSN Search nieda verżjoni li kienet turi riżultati minn Looksmart mgħaquda ma’ dawk ta’ Inktomi, b’eċċezzjoni żgħira meta riżultati mill-websajt AltaVista intużaw minflok. Wara dan, Microsoft ġedded is-servizz sabiex tibda tipprovdi r-riżultati tiegħu innifsu, b’index li beda jiġi mbiddel kull ġimgħa, jew anke kuljum. Dan il-proċess dam sa’ Novembru 2004, fejn ġie mnieda bħala servizz beta, u ġie mniedi għal kollox fi Frar 2005. Is-servizz tat-tfittxija għall-istampi kien ipprovdut mill-kumpanija Picsearch. Is-servizz tat-tfittxija tal-Microsoft sar ukoll disponibbli għal search engines oħra, bl-għan li jiżdied is-sehem fis-suq.

Windows Live Search

L-ewwel verżjoni beta ta’ Windows Live Search ġiet imnedija fit-8 ta’ Marzu 2006, bħala sostituzzjoni għal MSN Search. Is-search engine il-ġdida issa kienet tinkludi tabs li kienu jinkludu Web, aħbarijiet, stampi, mużika, desktop, lokali, kif ukoll is-servizz tal-Microsoft Encarta.

Fil-bidla minn MSN Search għal Windows Live Search, Microsoft waqfet tuża s-servizz ta’ Picsearch għat-tfittxija tal-istampi, u minflok bdiet tuża s-servizz ġdid tagħha.

Live Search

Nhar il-21 ta’ Marzu, 2007, Microsoft ħabbret li kienet se tissepara s-servizz tat-tfittxija mill-bqija tas-servizzi tal-Windows Live. L-isem inbidel għal Live Search, u l-paġna tal-ftuħ ħadet dehra ġdida, bi stampa li bdiet tinbidel kuljum.

Fil-perjodu tas-servizz ta’ Live Search, saru diversi tibiliet, fejn fost oħrajn waqfu s-servizzi ta’ Live Search Books, u Live Search Academic, li ġew integrati fit-tfittxijiet ġenerali. Ftit wara, Windows Live Expo ukoll waqaf jopera mill-31 ta’ Lulju, 2008. Live search Macros, servizz li bih il-konsumatur seta’ joħloq search engine personaliżżata, ukoll twaqqaf, kif twaqqaf ukoll is-servizz ta’ Live Product Upload, Live Search Products, u Live Search QnA, b’tal-aħħar isir MSN QnA, li iżda twaqqaf ukoll nhar il-21 ta’ Mejju, 2009.

Microsoft għarfet li l-isem “Live” kien għadu joħloq konfużjoni fis-suq, u li għalekk l-isem kellu jinbidel kompletament. F’kampanja sabiex tinħoloq identita’ ġdida għas-servizz tat-tfittxija tal-Microsoft, Live Search sar uffiċjalment Bing, nhar it-3 ta’ Ġunju, 2009.

Nhar id-29 ta’ Lulju, 2009 Microsoft u Yahoo ħabbru ftiehim ta’ 10 snin, li fih is-search engine tal-Yahoo issir dik ta’ Bing. Minn naħa l-oħra, Yahoo iżżomm 88% tal-profitt mill-bejgħ kollu fuq is-search engine fl-ewwel ħames snin tal-ftiehim, filwaqt li jkollha d-dritt tirreklama fuq xi websajts tal-Microsoft. Yahoo se żżomm id-dehra li dejjem kellha, pero’ eventwalment se jibda’ jidher il-logo ta’ Bing minflok.

Sehem fis-suq

Qabel it-tnedija ta’ Bing, is-sehem tas-search engines tal-Microsoft kienu neżlin b’rata mgħaġġla. Minn meta ġie mniedi Bing, is-sehem mhux talli waqaf, iżda beda’ tiela, speċjalment fl-iStati Uniti, fejn Bing sab postu fit-tielet post wara Google u Yahoo bħala l-aktar search engine użat. Minn Mejju 2009, fejn is-search engine kellha 8% tas-suq, fi Frar telgħet għal 11.5%, skont il-figuri maħruġa minn ComScore.
